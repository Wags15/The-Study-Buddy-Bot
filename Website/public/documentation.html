    +eyes

    - Reminds you to look away from your screen every 20 minutes, following the 20-20-20 rule for eye strain.

    - Type the command again to cancel it
    +pomodoro

    - Method designed to increase productivity and manage your time. It involves work periods of 25 minutes each, followed by a 5 minute break. Every fifth work period is 30 minutes.

    - Type the command again to cancel it
    +todoAdd

    - Add reminders, chores, or important notes to a list

    - For example, +todoAdd homework will add homework to your personalized list
    +todoView

    - View reminders, chores, or important notes from your personalized list
    +todoDelete

    - Remove items from your personalized list

    - For example, +todoDelete 1 will remove the first item in your list
    +gaming

    - Updates your status to gaming mode

    - Frequent reminders to take breaks and stretch

    - Type the command again to cancel it
    +water

    - Sends hourly reminders to stay hydrated

    - Type the command again to cancel it
    +bottle

    - Keep track of your daily hydration. Type this command to add to your total daily water bottle count.

    - Bottle count resets daily
    +studying?

    - Checks the status of whether any specified Discord user is in +pomodoro mode

    - For example, +studying? @Ryan will return whether Ryan is currently studying
    +meditation

    - Study Buddy Bot will suggest a random meditation based video from a varying list

    - To recieve a new suggestion, type +meditation again to prompt a new video
    +music

    - Study Buddy Bot will suggest a random music video from a list with varying genres

    - To recieve a new suggestion, type +music again to prompt a new song
